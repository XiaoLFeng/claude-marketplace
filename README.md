# Claude Marketplace

智能工作流管理助手 - 为 Claude Code 提供计划、待办、记忆的完整管理方案，支持批量操作和作用域隔离。

## 项目简介

本项目包含了一系列用于增强 Claude Code 工作流管理的插件和技能：

- **llm-memory**: 智能工作流管理插件，提供计划、待办事项和记忆的完整管理方案

## 功能特性

- 📋 计划管理：支持创建、查询和管理项目计划
- ✅ 待办事项：高效的任务追踪和管理
- 🧠 记忆功能：智能存储和检索工作上下文
- 🔄 批量操作：支持批量处理任务
- 🎯 作用域隔离：保持不同项目间的独立性

## 快速开始

详细的使用说明请参考各个子项目的文档。

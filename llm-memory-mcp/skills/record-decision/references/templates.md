# 记忆内容模板

## 技术决策模板

```markdown
## 背景
[为什么需要做这个决策，遇到了什么问题]

## 选项对比
| 方案 | 优点 | 缺点 |
|------|------|------|
| 方案A | ... | ... |
| 方案B | ... | ... |

## 决策
[最终选择的方案]

## 理由
1. [原因1]
2. [原因2]
3. [原因3]

## 影响
- [对项目的影响]
- [需要注意的地方]

## 相关
- [相关文档/链接]
```

## Bug 修复模板

```markdown
## 问题描述
[出现了什么问题，如何复现]

## 影响范围
[影响了哪些功能/用户]

## 原因分析
[根本原因是什么]

## 解决方案
[如何解决的]

## 代码变更
[主要修改了哪些文件]

## 预防措施
[如何避免再次发生]

## 相关
- [相关 issue/PR]
```
